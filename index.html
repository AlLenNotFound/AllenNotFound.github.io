About

I am a Master's student in High Performance Computing at Politecnico di Milano. My research interests include privacy and memorization in large language models, agent security, high-performance computing, and numerical methods for scientific computing.

I am particularly interested in understanding how machine learning systems memorize, leak, and expose private information, as well as developing reliable evaluation methods for privacy and security risks in modern AI systems.

Research Interests

  • Privacy and memorization in large language models
  • Agent security and privacy leakage through reasoning and tool interaction
  • Machine learning security and privacy evaluation
  • High-performance computing and parallel numerical methods
  • Numerical methods for scientific computing

Selected Projects

Semantic Similarity Based Reconstruction Evaluation
Designed an embedding-based evaluation framework for measuring semantic leakage in reconstructed images. The project studies whether reconstructed samples preserve semantically meaningful information beyond pixel-level similarity.
CUDA-based Parallel Convolution
Implemented optimized 2D convolution kernels using shared-memory tiling and evaluated GPU performance under different mask sizes and configurations.
Finite Element Methods with deal.II
Developed numerical solvers for partial differential equations, including Poisson, heat, and Stokes equations, using the deal.II finite element library.
MPI-based SIR Simulation
Implemented a parallel epidemic simulation using Cartesian domain decomposition, halo exchange, and Runge-Kutta time integration.
